# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `data-source.ts` file
3. Run `npm start` command

## Requiseicao post de event

```shell
{
  "title": "Visita COP",
  "description": "Evento para visitantes",
  "location": {
    "postalCode": "66000000",
    "numberHouse": "123",
    "complement": "Apto 3B"
  }
}
```
